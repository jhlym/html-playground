CSS Grid 속성

3개의 세로 열 트랙으로 구성. 하나의 열마다 최소 높이가 100px 넓이는 자동으로 늘어나는 구조

fr단위는 그리드 컨테이너에 남아 있는 사용 가능한 공간의 일정 비율을 나타냅니다.

MDN web doc 참조 
        
          .grid-1 {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            grid-gap: 10px;
            grid-auto-rows: minmax(100px, auto);
          }
        
      

      .grid-2 {
        display: grid;
        grid-template-columns: 1fr 1fr 1fr;
        grid-gap: 10px;
        grid-auto-rows: minmax(100px, auto);
      }
    

      .grid-3 {
        display: grid;
        grid-template-columns: 1fr 2fr 1fr;
        grid-gap: 10px;
        grid-auto-rows: minmax(100px, auto);
      }
    

      .grid-4 {
        display: grid;
        grid-template-columns: 20px repeat(6, 1fr) 20px;
        grid-gap: 10px;
        grid-auto-rows: minmax(100px, auto);
      }
